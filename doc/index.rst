********
EarthSim
********

**Short description of the point of this project**

Project is an `open-source
<https://github.com/pyviz/EarthSim/blob/master/LICENSE>`_ Python
library for leveraging the SciPy and PyViz ecosystem to work with
environmental simulations.

The `Getting Started <getting_started>`_ section provides an overview
of materials to familiarize with before getting started with EarthSim.

The `User Guide <user_guide>`_ provides examples to use drawing tools
to annotate data, running simulations and generate, visualize and
analyze different kinds of simulation output and meshes.

Please feel free to report `issues
<https://github.com/pyviz/earthsim/issues>`_ or `contribute code
<https://help.github.com/articles/about-pull-requests>`_.
